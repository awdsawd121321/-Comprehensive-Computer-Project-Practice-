# 25维面部行为特征逐维语义说明


## 1. 总体结构

25 维特征按如下顺序组成：

- 第 `0` 到 `14` 维：15个动作单元（AU）
- 第 `15` 到 `16` 维：2个情感维度（Valence, Arousal）
- 第 `17` 到 `24` 维：8个表情类别概率（Expression Probabilities）

## 4.文件格式要求

评测输出文件：

```text
prediction_emotion.npy
```

数组形状：

```text
[N, K, T, 25]
```

其中：

- `N`：样本数
- `K`：每个样本的候选序列数，默认 `10`
- `T`：时间长度，默认 `750`
- `25`：本文档定义的统一面部行为特征维度



- `58` 维 `3D_FV_files` 为可选输入特征，不属于统一评测输出格式，因此本文件不对其逐维语义做强制要求。
