## 第一步：生成预测文件

使用 val 数据集作为输入，模型生成 `prediction_emotion.npy`：

```bash
python -m face_driving.inference.generate_prediction \
    --checkpoint face_driving/checkpoints_emotion/best.pt \
    --data-root . \
    --split val \
    --index-csv val_package/perfrdiff_eval_pack/person_specific_val.csv \
    --diversity-strategy diverse \
    --smooth-window 3 \
    --var-alpha 1.3
```

**参数说明：**
- `--checkpoint`：模型权重路径
- `--data-root`：数据集根目录（包含 val/ 文件夹）
- `--index-csv`：官方样本顺序文件
- `--diversity-strategy diverse`：多样性增强推理策略（生成 K=10 条候选）
- `--smooth-window 3`：时序平滑窗口
- `--var-alpha 1.3`：方差放大系数

**输出：** `prediction_emotion.npy`，shape `[1086, 10, 750, 25]`

预计耗时：约 5 分钟（GPU）。

## 第二步：运行官方评测脚本

```bash
python val_package/perfrdiff_eval_pack/eval_emotion_metrics.py \
    --data-root . \
    --split val \
    --index-csv val_package/perfrdiff_eval_pack/person_specific_val.csv \
    --neighbor-matrix val_package/perfrdiff_eval_pack/person_specific_masked_neighbour_emotion_val.npy \
    --prediction prediction_emotion.npy \
    --metrics frcorr,frcorr_star,frdist,frdiv,frdvs,frvar,frsyn \
    --output-json results.json
```

## 模型输入输出说明

**输入：**
- Speaker 音频：`val/Audio_files/*.wav`（16kHz mono）
- Speaker 情绪条件：`val/Emotion/*.csv`（提取均值作为 10D 条件向量）

**输出：**
- 25 维面部行为特征序列：
  - dim 0-14：15 个 Action Units（AU）
  - dim 15-16：Valence / Arousal
  - dim 17-24：8 个表情概率

**模型架构：**
- 音频编码器：Chinese HuBERT Base（冻结）
- Transformer 解码器：4 层，d_model=256
- 条件注入：FiLM（Feature-wise Linear Modulation）
- 推理策略：分层多样性（Tier 1 低噪声保质量 + Tier 2/3 条件偏移增多样性）
